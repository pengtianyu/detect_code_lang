detect_code_lang
================

这是一个自动识别代码片段所使用的编程语言的项目。
他非常简单，但比较有效。起码在我的项目中是这样。
现在他支持10种语言(详见支持语言列表)。如果你想
要支持更多的语言，只需要添加要支持的语言的特征
罢了。

如果我非常有空的话，我可能会维护一下他。

用法：
  这个程序使用javascript实现，所以他仅仅是一个网
  页罢了。你只需要在装有浏览器的任何电脑上，简单
  的打开index.html即可。没有任何更多的需求。
